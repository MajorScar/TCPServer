1-To run basically open terminal and go to directory where the file is located. Make sure you go inside /src folder. 
2-Then complile the project using those two commands: 
javac TCPServer.java
javac TCPClient.java
3-Now that the porject is compiled you need to run the server using the following command:
java TCPServer
4-Now open a new terminal making sure you go to the correct directory in the new terminal then run the the TCPClient using the following command:
java TCPClient
5- to have multiple clients repeat step 4 multiple terminals
